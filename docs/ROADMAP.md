# Kree AI Roadmap

The evolution of Kree from a smart assistant to a personal digital entity.

## V1: Foundation (Current Build)
*The Core Orchestrator*
- **Intelligence**: Gemini 2.5 Live Multimodal brain.
- **Voice**: Vosk STT for offline, Google Live for cloud.
- **Agents**: 
    - Initial Browser Agent (Playwright).
    - Basic File Controller.
- **Integration**: MCP (Model Context Protocol) in progress.
- **Distribution**: Packaged `.EXE` build system with auto-updater.

## V2: Local Empowerment
*Privacy & Multi-Step Logic*
- **Kree LLM**: Replace Gemini Live with a custom-tuned local Kree LLM (running via Ollama).
- **Filesystem Librarian**: Advanced RAG (Retrieval-Augmented Generation) for local documents.
- **Task Chaining**: Ability to execute multi-step plans (e.g., "Search for a recipe, save it as a PDF, and email it to my wife").
- **Optimized UI**: Glassmorphism 2.0 dashboard with deeper OS integration.

## V3: Cognitive Evolution
*Habits & Personalization*
- **Habit Learning**: Kree learns user routines and suggests automations proactively.
- **Personalized AI**: Fine-tuned responses based on the user's specific writing style and history.
- **Hybrid Model**: Seamlessly switches between local and cloud models based on task complexity and privacy needs.
- **Expanded Ecosystem**: Support for smart home (IoT) integration.

## V4: Sovereign Intelligence
*Scaling & Ecosystem*
- **Kree-Core LLM**: Full custom LLM trained from scratch for low-latency assistant tasks.
- **Premium Tier**: Subscription model (₹299/month) for advanced cloud features and priority support.
- **Developer API**: Allow 3rd party apps to hook into Kree's intelligence.
- **Multi-Platform**: Extension to Linux (Ubuntu/Pop!_OS) and MacOS.
