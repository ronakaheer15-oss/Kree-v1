========================================================
  KREE AI — First Time Setup
========================================================

STEP 1: Run the app
  Double-click "Kree AI.exe" to launch.

STEP 2: Windows SmartScreen warning (first time only)
  Windows may show a "Windows protected your PC" warning.
  This is completely normal for new software that hasn't
  purchased a paid signing certificate yet.

  To continue:
    1. Click "More info"
    2. Click "Run anyway"

  This only happens once — Windows remembers your choice.

STEP 3: Enter your API key
  On first launch, Kree will ask for your Google Gemini
  API key. Get one free at:
    https://aistudio.google.com/apikey

STEP 4: Connect your phone (optional)
  Kree shows a QR code in the Connect tab.
  Scan it with your phone camera while on the same WiFi.
  This gives you a mobile companion app instantly.

========================================================
  TROUBLESHOOTING
========================================================

"The app won't start"
  → Make sure you extracted the ENTIRE zip folder, not
    just the .exe file. All files are required.

"No sound / microphone not working"
  → Check Windows Settings > Privacy > Microphone and
    make sure apps are allowed to use it.

"Phone can't connect"
  → Both devices must be on the same WiFi network.
  → Try disabling VPN if you have one running.
